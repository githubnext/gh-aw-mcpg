# gh-aw-mcpg
Github Agentic Workflows MCP Gateway

## Building

To build the CLI tool:

```bash
go build -o gh-aw-mcpg .
```

## Usage

Run the CLI tool:

```bash
./gh-aw-mcpg
```

Display version:

```bash
./gh-aw-mcpg version
```

Display help:

```bash
./gh-aw-mcpg help
```

## Development

This project requires Go 1.24.10 or later.

### Installing

```bash
go install github.com/githubnext/gh-aw-mcpg@latest
```
